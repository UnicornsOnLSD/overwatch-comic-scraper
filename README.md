# Overwatch Comic Scraper

A simple script to download all of the comics and short stories from Overwatch's media website (https://playoverwatch.com/en-us/media/stories/).

# Usage

The location is optional - the script will use the current directory by default.

`./overwatch-comic-scraper.py [location]`
